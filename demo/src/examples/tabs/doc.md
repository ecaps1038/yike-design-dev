### API

#### Tabs Props

| 参数           | 描述         | 类型     | 默认值 |
| -------------- | ------------ | -------- | ------ |
| value(v-model) | 选中项绑定值 | `string` | -      |

#### TabPane Props

| 参数  | 描述         | 类型                 | 默认值 |
| ----- | ------------ | -------------------- | ------ |
| name  | 标签的标识符 | `string` \| `number` | -      |
| label | 标签项的标题 | `string`             | -      |
